You are given an array of positive integers and a number (n). Try to find the greatest possible product of n elements in a consecutive series for the given array.

For example: array=[1,2,3,5,1,6,1] and n=2. The corresponding consecutive series are [1,2], [2,3], [3,5], [5,1], [1,6], [6,1], their products are 2, 6, 15, 5, 6, 6. The final result is 15.

Input: Two arguments. An array as a list of integers and a number n as an integer.

Output: The greatest possible product as an integer.

Example: 

checkio([1, 2, 3, 5, 1, 6, 1], 2) == 15
checkio([1, 3, 4, 5, 2, 6, 7, 2], 3) == 84

How it is used: This a simple exercise for working with a consecutive data types.

Precondition:
0 < len(array) ≤ 104
1 ≤ n ≤ 10
n ≤ len(array) all(1 ≤ x < 100 for x in array)
